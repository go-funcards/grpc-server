# gRPC Server

![License](https://img.shields.io/dub/l/vibe-d.svg)

## Installation

Use go get.

```shell
go get github.com/go-funcards/grpc-server
```

Then import **grpc-server** package into your own code.

```go
import "github.com/go-funcards/grpc-server"
```
